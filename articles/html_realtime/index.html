HTML編集サイトを作ってみた

2023/11/24

ここをクリックしてアクセスできます！

製作目的

一般的なHTMLの編集に用いる開発環境(Visual Studio Codeなど)では、見栄えの確認のために ファイルを保存して開く という手間が発生することがあります。

この手間を軽減するために、HTMLでコードを編集すると、その結果がすぐに反映されるシステムを作りました。

このHTMLタグはこの使い方で合ってるのかな？ といった確認用途や、簡単なHTML作品製作の用途などに使用できると思います。

使用方法

画面の左半分の枠内にHTMLを書くと、リアルタイムに 画面右側で結果が表示されます。

あとは直感的に使えるかと思います。説明をサボったわけではありません。

今後の課題

  • 書いたHTML (ソースコード) を出力する機能がない (現状、ソースコードを出力するには、メモ帳などにコピー・ペーストする必要があります)
  • HTMLのエラーを自動で見つける機能がない (この点では一般の開発環境には劣ります…)
  • たまにバグが起こります (あくまでも学生が作ったものなので許してください)
  • HTML編集システムのネーミングセンスがない
